# iChat - Secure Messenger

iChat is a real-time, secure messaging web application built with a modern full-stack approach. It features private user authentication and an encrypted-style messaging environment.

## 🚀 Live Demo
[View Live Site on Netlify](https://ichat2.netlify.app)

## ✨ Features
* **Real-time Messaging:** Powered by Firebase Firestore for instant message delivery.
* **Secure Auth:** Anonymous and username-based authentication logic.
* **Responsive UI:** Built with Tailwind CSS, optimized for mobile and desktop viewing.
* **Moderation Ready:** Includes a "Report Harassment" feature to keep the environment safe.
* **Emoji Support:** Integrated emoji picker for better user expression.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+)
* **Backend/Database:** Firebase Firestore
* **Authentication:** Firebase Auth
* **Hosting:** Netlify (Continuous Deployment via GitHub)

## 🔒 Security Measures
* **Environment Variables:** Sensitive Firebase API keys are protected using Netlify Environment Variables.
* **Security Rules:** Firestore rules ensure that only authenticated users can read or write data.
* **Privacy:** Designed to ensure chats are private and hidden from the public.
